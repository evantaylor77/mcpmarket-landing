Frequently asked questions

What are MCP servers?

MCP (Model Context Protocol) servers extend AI-powered IDEs like Cursor and Claude Code with additional tools for file operations, web search, database access, and more.

Is MCPMarket free?

Yes! MCPMarket is completely free and open-source software distributed under the MIT license.

Which package managers are supported?

MCPMarket supports npm, pip, go, and uvx package managers for installing MCP servers.

How do I add custom servers?

Use the "Add Custom Server" feature to add any MCP server with full configuration for installation and IDE setup.
